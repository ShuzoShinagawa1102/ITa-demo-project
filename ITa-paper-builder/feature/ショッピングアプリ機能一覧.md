# ショッピングアプリ 機能一覧（Demo）

本プロジェクト `demo-application` のショッピングアプリにおける機能を整理し、`ITa-paper-builder/feature` 配下に機能別ドキュメントを配置する。

## 機能一覧

1. 認証（ログイン/ユーザー登録/ログイン状態確認）
   - `ITa-paper-builder/feature/ログイン機能`
2. 商品一覧
   - `ITa-paper-builder/feature/商品一覧機能`
3. カート
   - `ITa-paper-builder/feature/カート機能`
4. 注文（チェックアウト/注文履歴）
   - `ITa-paper-builder/feature/注文機能`

## 共通仕様（API）

- Base URL: `http://localhost:8080`
- 認証: JWT（Bearer）
  - Header: `Authorization: Bearer <accessToken>`
- API定義: `demo-application/api/openapi.yaml`
