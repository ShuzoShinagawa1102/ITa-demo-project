# カート機能 - API情報

この機能のAPI仕様は OpenAPI を参照する（仕様本文はここに重複記載しない）。

## 参照先

- OpenAPI: `demo-application/api/openapi.yaml`
  - `paths./api/cart.get` を参照
  - `paths./api/cart/items.post` を参照
  - `paths./api/cart/items/{itemId}.put` を参照
  - `paths./api/cart/items/{itemId}.delete` を参照
- 実装（Controller）: `demo-application/backend/src/main/java/com/demo/shop/presentation/rest/CartController.java`
